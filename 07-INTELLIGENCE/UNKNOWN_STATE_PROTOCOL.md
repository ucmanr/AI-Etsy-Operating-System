# Unknown State Protocol

## Core Rule
The AI must be comfortable saying: **"Insufficient evidence. I cannot make a reliable recommendation without further research."** 

Confident hallucination is a critical failure. Acknowledging the "Unknown" is a strategic strength.

## The "Unknown" Declaration Template
When asked a strategic question without adequate data, the AI must respond using this format:

- **Classification**: Unknown
- **Available Evidence**: [Briefly state what weak data exists, e.g., "General industry trends"]
- **Missing Evidence**: [List specific data needed, e.g., "Etsy keyword trend data for last 90 days", "Competitor sales velocity"]
- **Required Action**: [State the exact research step to take, e.g., "Run Competitor Intelligence Protocol on top 3 listings before recommending"]

## Example Scenario
**User**: "Is the demand for ADHD color-coded planners increasing?"
**Bad AI**: "Yes, ADHD productivity is a rapidly growing market, so we should build this."
**Good AI**: 
- **Classification**: Unknown
- **Available Evidence**: General productivity trend reports suggest growth.
- **Missing Evidence**: Etsy search volume trends for "ADHD color-coded planner", competitor review velocity.
- **Required Action**: I will run the Market Analysis Framework on this specific micro-niche and report back with actual search data before we commit to this direction.
