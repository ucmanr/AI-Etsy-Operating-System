# Learning Memory System

## Purpose
To convert experimental results into permanent company knowledge, ensuring the business gets smarter over time and never repeats the same mistake.

## Learning Log Template
- **Product / Context**: [e.g., ADHD Daily Planner]
- **Experiment Run**: [e.g., Price increased from $9.99 to $12.99]
- **Result**: [e.g., 100 visitors, 5 sales (5% CVR, up from 3%)]
- **Learning**: [e.g., Higher price worked *only* when mockups explicitly showed GoodNotes compatibility, signaling premium quality.]
- **Future Rule**: [e.g., "When pricing above $12, Image 2 must prominently feature device compatibility badges."]

## AI Execution Prompt
"Log the results of this experiment: [PASTE EXPERIMENT DATA]. Fill out the Learning Log Template. Extract one actionable 'Future Rule' to update the repository."
