# SOP: Create Product

## Purpose
Create a digital product ready for Etsy.

## Inputs

- Customer problem
- Product idea
- Validation evidence

## Outputs

- Finished product
- Listing assets
- QA complete

## Steps

1. Define problem
2. Create outline
3. Design assets
4. QA
5. Export files
6. Prepare listing

## Done Criteria

- Files exported
- QA passed
- Listing ready
