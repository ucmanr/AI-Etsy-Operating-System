# Design QA Checklist

## Purpose
The final gate before a product is exported and handed to the SEO/Listing team. **All boxes must be checked.**

## Pre-Export QA
- [ ] **Margins**: Minimum 0.5 inch on all sides.
- [ ] **Typography**: Max 2 fonts used. Body text is >= 11pt. Line height is >= 1.5.
- [ ] **Contrast**: All text passes WCAG AA contrast check against its background.
- [ ] **Cognitive Load**: No page has more than 5 distinct sections or more than 10 checkboxes.
- [ ] **Forgiveness**: The design includes a "catch-up" or flexible element (not rigidly time-bound).
- [ ] **File Integrity**: All links (if any) are tested. No broken images or missing fonts.
- [ ] **Bundle Readiness**: The file is named and structured so it can be easily combined with other products (e.g., consistent page sizing).

## Sign-off
- [ ] Reviewed by Product Manager Agent
- [ ] Approved for Export (PDF/PNG)
