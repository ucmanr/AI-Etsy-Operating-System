# Canva AI & Manual Generation Prompts

## Purpose
Standardized prompts to rapidly generate or assemble assets in Canva while adhering to the Design System.

## Prompt 1: Generating a Background/Layout
"Create a minimalist, calming digital planner page background. Use an off-white (#FAFAFA) base. Add subtle, light gray (#E2E8F0) dotted grid lines. Leave a 0.5-inch margin on all sides completely blank. The style should be clean, modern, and neurodivergent-friendly with zero visual clutter."

## Prompt 2: Generating UI Elements (via Canva Elements search)
"Search terms to use: 'minimalist rounded checkbox', 'soft pastel highlight', 'clean sans-serif text box', 'subtle shadow container'. Avoid: 'grunge', 'vintage', 'neon', 'complex floral'."

## Prompt 3: Assembling a "Daily Focus" Page
"Using the 'Daily Focus' Layout Pattern: 
1. Place a large 'Top 3 Priorities' section at the top with gentle checkboxes.
2. Add a 'Brain Dump' dotted-grid box in the middle.
3. Add a 'Dopamine Menu' section at the bottom. 
Use Inter or Poppins font. Ensure all text is dark slate (#2D3748) for high contrast."
