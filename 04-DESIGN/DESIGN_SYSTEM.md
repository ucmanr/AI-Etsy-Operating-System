# Design System: Core Foundation

## Purpose
The single source of truth for all visual design in the AI-Etsy-Operating-System. Ensures every product is instantly recognizable, highly functional, and reduces cognitive load for neurodivergent users.

## Core Philosophy
- **Calm over Clutter**: White space is an active design element, not empty space.
- **Function over Decoration**: Every line, color, and icon must serve a cognitive purpose.
- **Forgiving Design**: The layout must guide the user gently, without inducing guilt or overwhelm for missed tasks.

## Spacing System (8pt Grid)
- **XS**: 8px (tight grouping, e.g., checkbox to text)
- **S**: 16px (standard element spacing)
- **M**: 24px (section separation)
- **L**: 32px (major block separation)
- **XL**: 48px+ (page margins, breathing room)

## Grid & Layout
- All digital planners/trackers must be designed on a standard **US Letter (8.5 x 11 in)** or **A4** canvas.
- Margins: Minimum 0.5 inch (36px) on all sides to prevent printer cutoff and provide visual boundaries.
