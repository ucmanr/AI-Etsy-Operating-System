# Typography Guide

## Purpose
To ensure maximum legibility and reduce visual stress for users with ADHD, dyslexia, or visual processing sensitivities.

## Font Selection Rules
- **Primary Font**: Must be a clean, highly legible Sans-Serif (e.g., Inter, Poppins, Open Sans, or Lexend).
- **Secondary/Accent Font**: Maximum one additional font for headings (e.g., a soft, rounded sans-serif like Quicksand). *Never use decorative, cursive, or highly stylized fonts for body text.*

## Hierarchy & Sizing (Digital PDF/Goodnotes)
- **Page Title (H1)**: 24pt - 28pt, Bold, High Contrast
- **Section Header (H2)**: 16pt - 18pt, Semi-Bold
- **Body Text / Prompts (H3)**: 12pt - 14pt, Regular
- **Micro-copy / Labels**: 10pt - 11pt, Regular, slightly muted color (but still passing contrast checks)

## Strict Constraints
- Line height must be at least 1.5x the font size to prevent text from feeling "cramped."
- Avoid ALL CAPS for paragraphs; use Title Case or Sentence case for better word-shape recognition.
