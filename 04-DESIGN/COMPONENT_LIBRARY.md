# Component Library

## Purpose
A catalog of reusable, pre-designed UI elements to speed up production and ensure consistency.

## Core Components
1. **The "Gentle Checkbox"**: A rounded square (4x4mm) with a soft border. When checked, it fills with the muted success color (`#68D391`).
2. **The "Dopamine Menu" Box**: A distinct, slightly shaded container divided into "Quick Wins" (<5 mins), "Medium Effort" (5-15 mins), and "Rewards".
3. **The "Brain Dump" Zone**: A large, dotted-grid or lightly lined area with a subtle, encouraging header ("Get it out of your head").
4. **The "Priority Star"**: A simple, clean icon used to denote the #1 task of the day.
5. **Progress Bar**: A simple, segmented bar (e.g., 4 blocks) to visually track completion of a multi-step task without using intimidating percentages.

## Usage Rule
Never redesign these components from scratch. Copy, paste, and adapt from the `PRODUCT_TEMPLATES` folder.
