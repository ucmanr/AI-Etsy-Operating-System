# Color System

## Purpose
To create a calming, focused environment that minimizes visual noise and anxiety, while maintaining strict accessibility standards.

## Core Palette (The "Calm Focus" Theme)
- **Background**: `#FAFAFA` (Off-white, reduces eye strain compared to pure `#FFFFFF`)
- **Primary Text**: `#2D3748` (Dark slate, high contrast but softer than pure black)
- **Secondary Text**: `#718096` (Muted gray for labels, hints, and completed items)
- **Accent / Action**: `#4A5568` or a muted, calming color like `#6B8E9B` (Soft teal) or `#9B8A7B` (Warm taupe). *Avoid bright reds, neon greens, or jarring yellows.*

## Functional Colors
- **Success/Completed**: `#68D391` (Soft, muted green)
- **Warning/Attention**: `#F6AD55` (Soft, muted orange - *never use alarming red for routine tasks*)

## Strict Constraints
- All text-to-background combinations must pass WCAG AA contrast ratio (4.5:1 minimum).
- Do not use color as the *only* indicator of status (e.g., a completed task should have a checkmark AND a color change).
