# Launch Checklist

## Purpose
The final, non-negotiable gate before a listing goes live. Prevents costly, embarrassing, or conversion-killing mistakes.

## Pre-Launch QA
- [ ] **SEO**: Title is under 140 chars, primary keyword is front-loaded. Exactly 13 tags are filled.
- [ ] **Attributes**: All 5 relevant Etsy attributes are selected.
- [ ] **Pricing**: Price is set correctly. Sale price (if applicable) is scheduled.
- [ ] **Files**: Correct, high-quality PDF/PNG files are attached. File names are clean (e.g., `ADHD-Daily-Planner-2024.pdf`, not `final_final_v3.pdf`).
- [ ] **Mockups**: All 10 images are uploaded in the correct sequence. Text is readable on mobile.
- [ ] **Description**: Copy follows the Listing Copy Framework. No broken links or formatting errors.
- [ ] **Inventory**: Set to 999 (for digital) or appropriate quantity.
- [ ] **Category**: Placed in the most specific, accurate Etsy category.

## Sign-off
- [ ] Reviewed by Marketing Agent
- [ ] Approved for Publishing
