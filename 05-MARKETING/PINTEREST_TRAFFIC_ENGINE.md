# Pinterest Traffic Engine

## Purpose
To systematically drive high-intent, external traffic to Etsy listings. Pinterest is a visual search engine, making it the #1 organic traffic source for digital planners.

## The "3-Pin Rule" Per Listing
For every new Etsy listing, create 3 distinct Pin designs:
1. **The "Aesthetic Setup" Pin**: Product displayed beautifully on an iPad/desk. Text overlay: "The planner that finally works for my ADHD brain."
2. **The "Problem/Solution" Pin**: Split screen. Top: "Overwhelmed by rigid planners?" Bottom: "Try this flexible, dopamine-friendly alternative."
3. **The "Feature Highlight" Pin**: Close-up video or carousel showing a specific feature (e.g., "How to use the Dopamine Menu").

## Pinterest SEO Rules
- **Pin Title**: Include primary keyword + benefit (e.g., "ADHD Daily Planner for Goodnotes | Reduce Overwhelm").
- **Pin Description**: 2-3 sentences naturally incorporating 3-5 long-tail keywords. Include a clear Call to Action (e.g., "Click to get instant access on Etsy").
- **Board Strategy**: Pin to 3-5 relevant, highly specific boards (e.g., "ADHD Productivity Hacks", "Digital Planner Setups").

## AI Execution Prompt
"Generate 3 distinct Pinterest Pin concepts for '[PRODUCT NAME]'. For each, provide: 1) Visual description, 2) Exact text overlay (max 8 words), 3) SEO-optimized Pin Title, 4) SEO-optimized Pin Description with a CTA."
